# 🤖 Movable AI Assistant

A hardware-based AI assistant capable of understanding voice commands and performing real-time actions through a movable platform. This project integrates Artificial Intelligence with embedded systems to create an interactive system.

## 🚀 Features
- Voice recognition  
- Real-time response  
- Mobility (hardware control)  
- AI-based command execution  

## 🛠️ Technologies Used
- Python  
- Machine Learning  
- Arduino  
- Sensors  

## ⚙️ How It Works
1. Captures voice input  
2. Processes using AI  
3. Sends commands to hardware  
4. Executes actions (movement/tasks)
